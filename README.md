# Hotel Management System (FYP)

- **Backend:** Django API in `backend/`
- **Frontend:** Next.js app in `frontend/`

From the repository root, `npm run dev` runs the frontend (see `package.json` scripts). Install JavaScript dependencies in `frontend/`:

```bash
cd frontend && npm install
cd .. && npm run dev
```

Django setup and run instructions are in `backend/` as applicable.
